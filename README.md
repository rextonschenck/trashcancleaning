# Ogden Garbage Can Cleaning

Marketing website for Ogden Garbage Can Cleaning — a garbage can cleaning service based in Ogden, Utah.

Plain HTML/CSS, no build step, no dependencies.

## File layout

```
index.html   # page content
styles.css   # styling
```

## Running locally

Just open `index.html` in a browser, or serve the folder with any static file server.

## Deploying

This is a static site — it can be hosted directly with GitHub Pages, Netlify, Vercel, or any static host.
